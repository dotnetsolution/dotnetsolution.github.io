How to show .NET version using dotnet command

From the Command Prompt on Windows or Terminal on Linux we can show the .NET version and information using dotnet command.

How to display .NET SDK version

To show the .NET SDK version we can use dotnet command with option version 

dotnet --version

Example 

C:\>dotnet --version
5.0.101

How to display .NET information

To show .NET information on your machine we can use dotnet command with option info 

dotnet --info
