Create ASP.NET Core Web Application using dotnet command

In this ASP.NET tutorial, we learn how to create a new ASP.NET core web application and run the application using dotnet command via Windows Command Prompt or Linux Terminal.

How to create ASP.NET core web application

From your Command Prompt, run the below command to create the new project. 

dotnet new webapp -o HelloApp --no-https

In the above command we have:

  • dotnet new to create a new project.
  • webapp to specify the ASP.NET Core Web Application.
  • -o HelloApp to specify the output location of the generated project.
  • --no-https to specify not enable HTTPS setting for your project.

How to run the ASP.NET core web application

From your Command Prompt, run the following command to navigate to the generated project folder. 

cd HelloApp

Then run the following command to run the ASP.NET application. 

dotnet run

Open the link http://localhost:5000/ on the browser to see the web page as following screenshot.

With the above steps you have finished creating a new ASP.NET project and run the application.
